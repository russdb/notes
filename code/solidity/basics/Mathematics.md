### mathematics

- Addition: x + y
- Subtraction: x - y,
- Multiplication: x * y
- Division: x / y
- Modulus / remainder: x % y (for example, 13 % 5 is 3, because if you divide 5 into 13, 3 is the remainder)

there is also an exponential operator.

`uint x = 5 ** 2` same as 5^2 =25
